**Cyber Guard - Automated Detection of Digital Arrest Scams**
Cyber Guard is an integrated web-based platform designed to combat the rising threat of digital arrest scams and other cybercrimes. The system provides a one-stop solution for user awareness, real-time scam detection, and streamlined reporting assistance.

🚀 Key Features
Awareness Module: Provides specialized cyber safety education through interactive stories, videos, and quizzes tailored for different age groups (Children, Teenagers, Adults, and Senior Citizens).

Real-Time Detection: Utilizes OpenAI's Whisper speech recognition model to convert voice input into text. It monitors for suspicious keywords like "police," "arrest," "FIR," or "warrant" to provide instant alerts to the user.

Reporting Module: Connects users directly to official government cybercrime portals, helps locate the nearest police stations, and assists in generating structured complaint reports in PDF format.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

Backend: Python

Database: MongoDB

AI Model: Whisper Speech-to-Text

Deployment: https://cyberguard-theta.vercel.app/

📂 Project Structure
The project is divided into three core pillars:
Awareness: Education and interactive quizzes to prevent scams.
Detection: Voice analysis and keyword detection for active threats.
Reporting: Integration with government portals and automated report generation.

🔧 Installation
Clone the repository:
git clone https://github.com/your-username/cyber-guard.git

Install the required dependencies:
pip install -r requirements.txt

Run the application:
python app.py
