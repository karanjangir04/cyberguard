CYBERGUARD - HTML/CSS/JS Project
=================================

FILES:
  index.html      → Home page
  awareness.html  → Awareness page (quiz, certificates, workshops)
  detection.html  → Real Time Detection page
  reporting.html  → Report Cyber Crime page
  casestudy.html  → Case Studies (use ?type=kids / teens / adult / oldage)
  style.css       → Shared stylesheet (include in all pages)

IMAGES FOLDER:
  Place these images in the /images/ folder:
  - banner1.jpg, banner2.jpg, banner3.jpg, banner4.jpg  (carousel banners)
  - cdac_logo.jpg   (CDAC logo for top navbar)
  - isea_logo.jpg   (ISEA logo for top navbar)

HOW TO USE:
  Simply open index.html in any browser.
  No build tools, no npm, no React — pure HTML/CSS/JS!

CASE STUDY LINKS:
  casestudy.html?type=kids
  casestudy.html?type=teens
  casestudy.html?type=adult
  casestudy.html?type=oldage
